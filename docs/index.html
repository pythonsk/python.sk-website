Python je moderný programovací jazyk. Je univerzálny, poháňa weby, ale aj robotov. Je výborný pre vyučovanie programovania.

Komunita na slovensku je mladá a aktívna. Najdeš tu pomoc, kamarátov, alebo aj prácu. Vitaj medzi nami.

PyCon Slovakia

Slovenská Python konferencia! Každoročne sa tam stretne takmer 600 členov komunity. Pomôž nám s prípravami, propagáciou, skús si prihlásiť prednášku, či workshop. Sponzoruj nás, alebo len príď a uži si akciu!

Učíme s Hardvérom
